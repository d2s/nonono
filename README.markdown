## Requirements ##

* [Ruby][] version: 1.8.7+
* [rdiscount][] gem version: 1.6.8+
* [kicker][] gem version: 2.3.0+
* [mustache][] gem version: 0.12+
* [Sass][] version: 3.0.25+
* [Compass][] gem version: 0.10.6+
* [chronic][] gem version: 0.3.0+
* [Ramaze][] gem version: 2011.01.30+


Installing gems:
